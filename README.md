## How to Use

1. **Visit** [https://icybits-lab.github.io/WeatherFlow/](https://icybits-lab.github.io/WeatherFlow/)
2. **Search** for any city worldwide (e.g., Dhaka, London, Tokyo)
3. **Or click** "Use My Location" for your local weather
4. **Toggle** between metric/imperial units as preferred
5. **Switch** dark/light theme for comfortable viewing
6. **Explore** different map layers (temperature, precipitation, clouds, wind)
7. **Check** air quality and detailed pollutant information

## Technical Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: OpenWeatherMap (Current Weather, 5-day Forecast, Air Pollution, Weather Maps)
- **Maps**: Leaflet.js with OpenStreetMap tiles
- **Icons**: Font Awesome 6
- **Storage**: LocalStorage for user preferences
- **Hosting**: GitHub Pages

## Privacy

- No user data is collected or stored on servers
- Preferences (theme, units, recent searches) are stored locally in your browser
- Location data is only used when you explicitly click "Use My Location"
